HTML e Git

HTML não é uma linguagem de programação e sim de marcação que utiliza tags para assinalar cofigurações.

Git é um sistema de armazenamento e versionamento de códigos e projetos

CSS

CSS é uma linguagem de estilos (cuida da parte estética do texto ), é ele que da cor ao texto na web, muda a fonte e diversas outras coisas.

VSCode

O VSCode é um programa para escrever os codigos.

Tags Utilizadas

  • HTML

    • Ela vai estar presente em todos os documentos, e em todos eles essa tag deve iniciar e finalizar.

  • HEAD

    • Essa tag delimita o cabeçalho do documento.

  • title

    • Essa tag define o título da sua página, o nome que aparecerá na sua aba, janela ou guia

  • body

    • A tag que representa o corpo do documento.É nela que todos os elementos visíveis do seu site devem ser inseridos.

  • H1 ate H6

    • A tag H é de Título, sendo o menor h6, e o maior h1.
